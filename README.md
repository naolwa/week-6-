**Scenario:**

**Requirements:**

1. **List of Sensors:**

   * Soil moisture sensor
   * Temperature and humidity sensor
   * Light intensity sensor

   * Create a **Data Flow Diagram (DFD)** showing how sensor data is collected, processed, and analyzed by the AI system.
